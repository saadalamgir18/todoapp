## Getting Started

First, run the development server:

```bash
git clone https://github.com/saadalamgir18/todoapp.git
#then
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
